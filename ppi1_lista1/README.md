## Sobre o Projeto:
[cite_start]Este repositório contém a primeira lista de atividades da disciplina de PPI[cite: 3]. [cite_start]O objetivo foi praticar os fundamentos da Web, estrutura de pastas, tags HTML5, tabelas, formulários e estilização básica com CSS externo[cite: 13, 21, 25, 33, 75].

- [cite_start]**Atividade 2:** Explicação do protocolo HTTP[cite: 21].
- [cite_start]**Atividade 3:** Tabela de estados brasileiros (UFs)[cite: 25].
- [cite_start]**Atividade 4:** Currículo Profissional e Formulário de Contato[cite: 33, 52].
- [cite_start]**Atividade 5:** Estilização global via `style.css`[cite: 75].
- [cite_start]**Atividade 6:** Publicação via GitHub Pages[cite: 87].
